# Distance Learning Committee Operation

통합 플랫폼으로 Faculty Email Finder와 Course Modality DB를 관리합니다.

## 기능

### 📧 Faculty Email Finder
- 이름 기반 이메일 자동 매칭
- 교원 정보 엑셀 업로드 및 개별 입력

### 📚 Course Modality DB
- 강의 정보 엑셀 업로드 및 개별 입력/수정
- 검색 및 필터링
- Reason for Applying 영어 번역
- 전체/학기별 엑셀 다운로드
